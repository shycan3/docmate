## 슬라이드 배치 팁

- 첫 화면 이미지는 크게 배치하고, 주변 설명은 최소화한다.
- 결과 요약 이미지는 `신청 가능`, `위험 조건`, `할 일` 세 부분에 콜아웃을 붙이면 좋다.
- 원문 근거 이미지는 "자동화의 신뢰 문제를 어떻게 완화했는가"라는 질문에 답하는 화면으로 사용한다.
- 체크리스트 이미지는 사용자가 다음 행동을 바로 알 수 있다는 점을 강조한다.
- 히스토리 비교 이미지는 발표 말미에 "단일 공고 분석을 넘어 여러 기회를 비교한다"는 확장성을 보여준다.

## 주의할 점

- 현재 캡처는 로컬 데모 데이터와 샘플 공고를 기준으로 한다.
- 샘플 URL은 실제 기관 링크가 아니라 발표용 예시 링크다.
- 실제 출시 단계에서는 실공고 데이터, 개인정보 처리, 계정/배포/운영 안정화가 추가되어야 한다.
